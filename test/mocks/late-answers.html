    How to use the Late answers queue
    Awarded at 500 reputation

    This queue contains answers that were posted much later than when the question was asked. Reviewing these posts makes sure that questions maintain quality answers and problematic answers get removed.

    Here is some information on how to work through this queue:

    • Consider leaving a comment when appropriate.
    • Choose Looks OK if the post is fine as-is.
    • Edit the answer if you think you can improve it.
    • Delete answers that do not address the question at all, are link-only, or are incomprehensible.
    • Skip the task if you aren't sure which action to take.
    Read more about this queue in our help center.
